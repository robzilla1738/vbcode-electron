## Description

Vibe Codr brings the vibe-codr agent engine to a focused desktop workspace.
Open a project, choose the AI provider and model that fit the job, and work with
an agent that can understand files, run tools, review changes, and keep you in
control of every important action.

Use the integrated transcript, project and session navigation, Git review,
persistent terminal, changed-file diff, jobs, and approval surfaces without
leaving the conversation. Vibe Codr is local-first: you choose the projects and
external AI services it can use, and the app does not operate its own analytics,
advertising, telemetry, or remote crash-reporting service.

Key features:

- Broad AI provider and model support through the vibe-codr engine
- Project-aware conversations with files, mentions, images, and attachments
- Built-in Git status, diff review, terminal, jobs, and session inspection
- Explicit permission and approval controls for consequential actions
- Persistent project and chat organization with searchable command workflows
- Multiple themes, density modes, keyboard shortcuts, and responsive layouts
- Local-first settings and session data

AI provider accounts, API keys, or local model runtimes may be required for AI
features. Use of external providers is subject to their terms and privacy
policies.
